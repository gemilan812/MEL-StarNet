# Industry Bottleneck Investment Research Skill

## 项目简介

这是一个用于投资研究场景的 Claude/Codex Skill，目标是帮助使用者从产业趋势、供需错配、产业链瓶颈、财报验证、资金扩散和催化剂中提炼结构化研究框架。

本 Skill 不用于预测股价，不提供买入或卖出建议，不承诺收益。它的核心作用是把新闻、财报、公告、研报、推文、产业数据等信息整理成可验证、可迁移、可证伪的研究框架。

适用范围包括但不限于：

- A 股、港股、美股、ETF、日韩台市场和全球产业链映射
- 半导体、AI、算力、机器人、新能源、金属、化工、消费、医药、军工、游戏、白酒等行业
- 龙头财报超预期、上游涨价、供给短缺、技术路线变化、资金轮动、库存周期反转等主题

## 方法论来源说明

本 Skill 的方法论灵感部分参考了 X/Twitter 用户 [@aleabitoreddit](https://x.com/aleabitoreddit) 公开分享的产业链研究思路，尤其是以下研究视角：

1. 不只关注热门概念或单一个股，而是沿产业链寻找真正紧缺、不可替代或具备供给约束的环节。
2. 不只看终端龙头，而是观察龙头需求外溢后，上游材料、关键零部件、设备、封装、测试、能源、电力等环节是否存在重新定价机会。
3. 不只看新闻本身，而是判断新闻是否可能影响收入、利润、订单、价格、毛利率、产能利用率或市场预期。
4. 不只看个股逻辑，而是同时观察产业链传导、预期差、催化剂、财报验证和资金扩散。
5. 对小市值、高弹性、远期放量、客户认证、产能分配和技术路线变化保持更严格的证据要求。

本项目并未复制、搬运或代表该用户的原始推文内容，也不代表该用户本人观点、授权、认可或背书。Skill 中的框架是基于公开信息和方法论抽象整理后的通用研究工具，目标是提升研究过程的结构化程度和风险意识。

## 核心能力

使用该 Skill 后，模型会按以下流程进行分析：

1. 识别主线事件：技术突破、产业扩产、产品涨价、供给短缺、财报超预期、政策驱动、资金异动等。
2. 拆解产业链：原材料、核心设备、关键零部件、制造、封装/集成、渠道/客户、终端应用、替代技术和配套服务。
3. 寻找瓶颈环节：产能、扩产周期、技术壁垒、替代供应商、认证周期、客户绑定、产能分配和成品输出控制权。
4. 建立标的池：区分确定性龙头、瓶颈环节、高弹性标的、补涨扩散标的和观察名单。
5. 判断预期差：识别认知预期差、产业链预期差、财务预期差、技术预期差、产能预期差、估值预期差、区域预期差和资金预期差。
6. 跟踪催化剂：财报、订单、涨价、产能、客户认证、政策、行业会议、海外映射、ETF 资金流等。
7. 检查风险和证伪条件：防止概念炒作、相关性不足、估值透支、资金拥挤、技术路线切换和产业验证失败。

## 输出内容

默认输出包括：

- 一句话结论
- 事件与主线识别
- 产业链拆解表
- 核心标的分层
- 预期差分析
- 催化剂时间线
- 风险清单
- 研究结论

Skill 会强制区分：

- 明确事实
- 归纳逻辑
- 研究框架
- 风险与证伪条件

如果信息不足，输出中必须标注 `需要进一步验证`。如果标的相关性弱，必须降级为观察名单。若逻辑无法被数据验证，必须标注为 `概念性较强`。

## 使用提示

如果你希望模型使用该 Skill，请提供尽可能完整的信息：

- 产业主题或事件
- 已知核心标的
- 上游、中游、下游产业链信息
- 财报、公告、新闻、研报、推文或会议纪要
- 当前市场表现
- 关注市场范围
- 风险偏好
- 期望输出格式

信息越完整，输出越容易从“概念性观察”升级为“可验证研究框架”。
